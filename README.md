# 오픈소스 가이드

기업에서 오픈소스를 사용하면서 어떻게 하면 최대한의 가치를 창출할 수 있을지를 고민하고 있습니다. 조직 전체에 올바른 오픈소스 문화가 확산될 수 있도록 환경과 시스템을 구축하면서 경험한 사항을 토대로 오픈소스 가이드를 작성하고 있습니다.
